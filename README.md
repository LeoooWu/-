# 鲸邻助贷系统

#### 系统介绍
  鲸邻助贷系统是一款高性价比的独立部署解决方案，一次性买断无年费，价格仅为市场价1/4。支持不限账号，数据独立存储，安全可靠。功能包括营销管理（分销、渠道收单）、免费外呼中心（通话录音）、全流程数据统计（客户跟进、业绩分析），并支持定制开发。集成DeepSeek AI，可自动生成跟进记录、完善客户资料、智能分析签单意向及优化话术，提升销售效率。

##### 系统功能概述
* 不限使用账号数、支持真正的系统能+数据库均独立部署；
* 支持H5表单获客、支持员工自主拓客和营销、支持产品匹配以及匹配方案一键生成报告；
* 支持任务待办、支持短信提醒；
* 自带统计中心，支持公司业绩统计、做单业务统计、客户质量统计、客户预约统计、客户签单统计、客户进件统计、客户放款统计、客户回款统计等；
* 支持客户黑名单判断，支持基于deepseek的人工智能ai判断客户质量，预防员工误判；
* 支持做单管理，邀约/签单/进件/放款/回款/贷后管理;
* 自带呼叫中心，支持电话轮呼、通话统计、话术模板，支持人工智能自动分析销售话术、提炼销冠的话术并形成新的话术模板；
* 支持渠道接口对接，对接第三方运营平台、广告投放平台等；
* 支持各种条件下的客户流转规则配置；
* 支持员工跟进提醒、跟进方式自定义、人工智能自动填写跟进记录；
* 客户资料、做单资料等均可以自定义字段管理，支持AI人工智能自动完善客户资料；
* 支持自定义派单规则设置、固定设备/时间登录规则配置；
* 支持贷款产品与资方机构管理、产品匹配规则自定义、产品匹配快速生成贷款方案；
* 支持组织架构管理、角色权限管理、客户与产品标签管理等功能特色。

* 多场景业务支持：灵活支持多种业务模式，如电销、渠道合作和直销业务等，可根据业务类型进行优化配置，满足不同贷款机构的需求 。
* 高效协作与沟通：支持多人协作模式，实时同步业务进展，确保信息传递及时准确。同时，提供PC与移动端支持，方便员工随时随地处理业务 。
* 数据权限管理：提供精细的权限控制功能，可根据不同岗位的需求，精确设置数据访问权限，防止权限越权行为，保障数据安全 。
* 成本效益高：采用一次性付费终身免费使用的模式，价格仅为市场价的四分之一，无需每年支付续费，为企业节省了大量长期使用成本 。
* 灵活部署：支持独立部署，可根据企业需求选择不同的服务器和操作系统，确保系统的稳定性和安全性。同时，支持二次开发和定制，能够更好地满足企业的特殊需求 。
* 不限账号数：没有账号数量的限制，可根据企业需求灵活配置账号数量，更加符合中小型助贷企业的实际需求 。
* 丰富的功能：整合了100+高质量线索渠道，支持一键开启线索转化，极大地降低了营销成本。具备获客引流、产品分享、渠道收单、分销获客、产品推荐和方案匹配等丰富的营销获客功能 。

#### 鲸邻助贷系统功能模块介绍

##### 工作台
系统版本号、员工账号信息、待办任务、业绩统计（成功签单数、成功进件数、放款金额、回款金额）、客户质量报表、待跟进客户、业务排行榜、通话排行榜；


##### 线索管理
线索列表、撞库记录、派单记录；


##### 客户管理
新客池、跟进中、已上门、重点客户、成交客户、撞库客户、协同客户、全部客户、部门公海、公司公海、客户黑名单、跟进记录、客户签到、员工拓客；



##### 做单管理
预约管理、签单管理、进件管理、放款管理、回款管理；

##### 贷后管理
贷后列表、还款计划管理

##### 呼叫中心
轮呼管理、通话统计、通话记录、话术模板


##### 智能营销
获客渠道管理、H5产品获客、短信营销管理

##### 统计中心
公司业绩统计、做单业绩统计、客户质量统计、客户回款统计、新增客户统计；

##### 运营配置
业务规则配置、资料字段配置、自动派单配置、固定登录环境；

##### 系统管理
角色管理、部门管理、员工管理、通知公告、标签管理、字典管理、操作日志。
